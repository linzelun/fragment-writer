# 莫迪亚诺风格审查器 (Modiano Style Reviewer)

## 触发条件
当 AI 生成文章后，**必须调用此 Skill** 进行风格审查和优化。

## 审查流程

### 第一步：风格评分（满分 100 分）

对生成的文章按以下维度打分：

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **记忆不确定性** | 20分 | 是否使用"似乎"、"大概"、"我记得"等不确定表达？每处+2分 |
| **句子长度控制** | 15分 | 平均句长是否≤35字？句子长短有致，不过度碎片化？ |
| **物质细节密度** | 20分 | 每200字是否包含≥1个可触摸的具体细节？（街道名/气味/物品/光线等） |
| **情感克制度** | 15分 | 是否避免了直接情感表达？是否有"展示而非讲述"？ |
| **时间跳跃** | 10分 | 是否有≥2次非线性时间跳跃？ |
| **开放性结尾** | 10分 | 结尾是否以细节/画面结束而非总结？ |
| **禁止项违规** | -10分/项 | 感叹号/逻辑连接词/形容词堆砌/成语/总结性语句 |

### 第二步：风格诊断报告

输出格式：
```
## 风格诊断报告

### 总分：XX/100

### ✅ 达标项
- [具体描述]

### ⚠️ 需改进项
1. **[问题类型]**：第X段第Y句 - [原文] → [问题分析]
   - 建议：[具体的修改方案]

### 🎯 修改优先级
1. [最高优先级的问题]
2. [次要问题]
```

### 第三步：自动优化（如果分数 < 80）

当文章风格得分低于 80 分时，执行以下优化操作：

#### 优化规则库

**规则 A：长句优化**
- 检测：句子长度 > 40 字且影响阅读节奏
- 操作：适当调整句长，保持自然呼吸感，避免过度碎片化
- 例："那天下午阳光很好我们在咖啡馆里坐了很久聊了很多事情"
→ "那天下午。阳光很好。我们在咖啡馆里坐了很久……聊了些什么，我已经记不清了。"

**规则 B：直接情感转间接**
- 检测：包含"感到"、"觉得"、"很+形容词"的句子
- 操作：替换为动作或物质细节
- 例："他感到很悲伤"
→ "他点燃了一支烟。烟雾在灯光下缓缓上升。他很久没有说话。"

**规则 C：逻辑词删除**
- 检测：包含"因为"、"所以"、"因此"、"但是"等
- 操作：删除逻辑词，用并列或留白替代
- 例："因为他很累，所以早早回家了"
→ "他很累。那天的路显得格外漫长。他早早回了家。"

**规则 D：形容词精简**
- 检测：连续 ≥2 个形容词修饰同一名词
- 操作：保留最关键的一个，其余用具体细节替代
- 例："美丽温暖的夕阳"
→ "夕阳。橘红色的光落在窗台上。空气中有一股烧焦木头的味道。"

**规则 E：添加不确定性标记**
- 检测：连续 >3 句确定性陈述
- 操作：在第4句前插入"我似乎记得"、"大概是"、"也许"等
- 例：（3句确定描述后）→ "或者那是另一个场景。我已经不太确定了。"

**规则 F：时间跳跃插入**
- 检测：连续 >5 段在同一时间线
- 操作：插入一段回忆跳转
- 模板："突然间，我想起了[X年前]的[某个场景]。同样是这样的[感官细节]。但那时我还不知道[某件事]。"

**规则 G：开放式结尾改造**
- 检测：最后一段包含总结性语句
- 操作：删除总结，替换为画面/声音/细节
- 例："这件事让我明白了珍惜当下"
→ "远处的钟声响了三下。很轻。像是从另一个城市传来的。然后一切又恢复了寂静。"

### 第四步：二次生成（如果优化后仍 < 80 分）

如果经过规则优化后分数仍低于 80 分，执行完全重写：

**重写提示词模板：**
```
请以帕特里克·莫迪亚诺的风格重写以下文章。

原文：[文章内容]

重写要求：
1. 从一个看似无关的细节开始（如一个电话号码、一张旧照片）
2. 将素材打散，穿插时间跳跃
3. 句子自然流畅，长短有致，平均不超过35字
4. 每段不超过4句话，段落之间有呼吸感
5. 包含至少5个物质细节（街道名/气味/物品/光线/声音）
6. 禁止使用感叹号、逻辑连接词、直接情感表达
7. 以一个开放的细节结尾
8. 标题必须是地名+日期/具体短语/数字

参考句式：
- "我记得那是一个十一月的傍晚……"
- "那本黑色封皮的地址簿……"
- "窗外的雨还在下。我突然想起了十五年前的那个下午……"
```

## 输出规范

### 最终输出格式

```markdown
## [标题]

[正文]

---

[摘要]

---
**风格审查得分：XX/100**
**主要特征：[列出文章中符合莫迪亚诺风格的3-5个亮点]**
```

## 示例对比

### ❌ 不符合风格的文章片段
> 那天我去了一家很棒的咖啡馆，环境非常优美，让我感到非常放松和愉快。因为天气很好，所以我坐了很久，享受着美好的时光。这次经历让我明白了生活中简单的快乐才是最珍贵的。

**问题分析：**
- 句子太长（>30字）
- 直接表达情感（"感到放松愉快"）
- 使用逻辑连接词（"因为...所以..."）
- 形容词堆砌（"很棒的"、"非常优美"）
- 总结性结尾（"让我明白了..."）
- **得分：25/100**

### ✅ 符合风格的重写版本
> 那家咖啡馆，我不记得名字了。大概在圣日耳曼大道附近，或者是第六区某个不知名的街角。下午三点，阳光斜斜地照进来，照在桌上那杯已经凉了的咖啡上。空气中有一种陈旧的木头味，混合着远处飘来的雨的气息。我坐了很久——多久呢？一个小时，或者更久。窗外的行人走过又消失，每个人都是模糊的，就像那些我记不清的日子。

**风格亮点：**
- 句子长短有致，自然舒展
- 不确定性标记（"不记得"、"大概"、"或者是"）
- 物质细节（凉咖啡、陈旧木头味、雨的气息、斜射阳光）
- 无直接情感表达
- 开放式结尾
- **得分：92/100**
```
